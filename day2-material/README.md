# SO-101 Day 2 Material

Standalone copy of the Day 2 lesson page for:

- Planning a LeRobot recording session
- Recording demonstration episodes
- Reviewing and curating a dataset
- Training a policy from the dataset

The main Vite app now also serves this material as a React route.

From the project root:

```bash
npm run dev
```

Then open:

```text
http://127.0.0.1:5173/day2
```

If Vite chooses another port, use the printed port and add `/day2`.
